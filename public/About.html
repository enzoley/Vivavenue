Principe du projet

Application permettant de réaliser des recherches précises sur le site d'annonce Vivastreet. Utilise les mots clés "and" et "or" afin de cibler la recherche plus précisément que la recherche classique. Présente les résultats sous forme de page HTML avec des liens cliquables.

Utilisation de l'application

Pour lancer une recherche il suffit de run le fichier Application.scala via sbt dans un terminal de commandes. Ensuite, il suffit de taper la requête grâce à des mots clés simples séparés par les mots clés "and" et "or".
